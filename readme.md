# WarNU_2

Aplikasi Android untuk [deskripsi singkat aplikasi, misal: manajemen inventaris, pemesanan, atau fitur utama aplikasi].

## Fitur Utama

- [✔️] Manajemen produk
- [✔️] Pemesanan dan pembayaran
- [✔️] Notifikasi
- [✔️] Profil pengguna
- [✔️] Navigasi intuitif

## Struktur Proyek

- `app/` - Modul utama aplikasi Android
  - `src/main/java/com/imersa/` - Source code aplikasi
  - `src/main/res/` - Resource (gambar, layout, dll)
  - `src/main/AndroidManifest.xml` - Manifest aplikasi
- `gradle/` - Konfigurasi Gradle
- `build.gradle.kts` & `settings.gradle.kts` - Konfigurasi build

## Cara Build & Jalankan

1. Pastikan sudah menginstall Android Studio & JDK.
2. Clone repository ini:
   ```
   git clone https://github.com/irsyadst/WarNU_2.git
   ```
3. Buka folder di Android Studio.
4. Klik "Run" untuk membuild dan menjalankan aplikasi di emulator/device.
